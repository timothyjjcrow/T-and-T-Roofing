# T & T Roofing Website

A modern, responsive website for T & T Roofing company based in Eureka, CA. The website showcases their services, projects, customer testimonials, and contact information.

## Features

- Responsive design that works on all devices
- Modern animations and interactive elements
- Image gallery with filtering functionality
- Testimonial slider
- Contact form with validation
- Smooth scrolling and navigation
- Mobile-friendly navigation menu

## File Structure

```
├── index.html              # Main HTML file
├── css/
│   └── styles.css          # Main stylesheet
├── js/
│   └── main.js             # JavaScript functionality
├── images/                 # Contains all images used on the website
└── README.md               # This file
```

## How to View the Website

1. Simply open the `index.html` file in any modern web browser.

## Required External Resources

The website uses the following external resources, which are loaded via CDN:

- Font Awesome 6.4.0 (for icons)
- Google Fonts (Montserrat and Raleway)
- jQuery 3.6.0
- AOS (Animate On Scroll) library

## Browser Compatibility

The website is compatible with all modern browsers:

- Chrome
- Firefox
- Safari
- Edge

## Customization

To customize the website:

1. Modify the content in `index.html`
2. Adjust styles in `css/styles.css`
3. Update functionality in `js/main.js`
4. Replace images in the `images/` directory

## Color Scheme

The website uses a modern color scheme defined in CSS variables:

- Primary: #2c3e50 (Dark blue)
- Secondary: #e74c3c (Red)
- Accent: #3498db (Blue)
- Light: #ecf0f1 (Light gray)

You can easily change these colors by modifying the CSS variables in the `:root` selector in the CSS file.

## For Developers

If you need to make significant changes to the website, here are some tips:

1. The website uses CSS Grid and Flexbox for layout
2. Animations are handled by both CSS and JavaScript
3. The responsive design breakpoints are:
   - Mobile: Up to 768px
   - Tablet: 768px to 991px
   - Desktop: 992px and above
4. Interactive elements like the gallery filter and testimonial slider are implemented in JavaScript
